This report details my approach on coming up with functions that identify and fix issues in the 
HR dataset (uncleaned dataset), and then use the cleaned data to calculate key HR metrics. 
The main focus is coming up with solutions using Python data structures such as 
lists, dictionaries, and sets, and how we applied loops and conditional logic to clean and analyse 
the dataset using pytest
I read the data using the helper function, which converted the dataset into a 2D list then cleaned 
the data by using functions to remove invalid entries and fix formatting issues. 
Lastly i then used cleaned dataset to calculate HR metrics such as retention, turnover, and averages 
across departments.
Overall, this project helped me turn raw employee data into meaningful insights.
